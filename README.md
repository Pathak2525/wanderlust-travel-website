# Wanderlust 🌍

Wanderlust is a comprehensive, full-stack web application designed for travel enthusiasts. It provides a dynamic platform for users to explore, list, and review travel properties and destinations around the globe. 

## 🚀 Features

* **Dynamic Listings:** Users can view, add, edit, and delete travel property listings.
* **Interactive Maps:** Integrated with Mapbox to display exact property locations on a beautifully rendered map.
* **Image Uploads:** Secure and efficient image uploading and storage for property photos using Cloudinary.
* **User Authentication:** Secure signup, login, and logout functionality.
* **Authorization:** Only property owners can edit or delete their listings, and only logged-in users can leave reviews.
* **Responsive Design:** Fully responsive UI built with Bootstrap, ensuring a seamless experience across desktop and mobile devices.

## 🛠️ Tech Stack

* **Frontend:** EJS (Embedded JavaScript templating), HTML5, CSS3, Bootstrap
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (with Mongoose ODM)
* **Third-party APIs & Services:** * **Mapbox API** (for Geocoding and Maps)
* **Cloudinary** (for Cloud Image Storage)
